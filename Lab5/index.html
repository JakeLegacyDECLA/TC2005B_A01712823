Preguntas HTML

  1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

    Internet es la estructura global de redes interconectadas que utilizan TCP/IP, mientras que la World Wide Web es un servicio que funciona sobre Internet.
    (Internet Society. (s.f.). What is the Internet? https://www.internetsociety.org/internet/what-is-the-internet/)

  2. ¿Cuáles son las partes de un URL?

    Protocolo, dominio, puerto, ruta, query string y fragmento.
    (Berners-Lee, T., Fielding, R., & Masinter, L. (2005). RFC 3986.)

  3. ¿Qué significa el código HTTP 200?

    Indica que la solicitud fue exitosa.
    (Fielding, R., Nottingham, M., & Reschke, J. (2022). RFC 9110.)

  4. ¿Qué significa deprecated en HTML5?

    Que ya no se recomienda su uso.
    (W3C. (2014). HTML5.)

Laboratorio 3 - CSS

  1. ¿Qué son las hojas de estilo en cascada (CSS)?

    Permiten definir la presentación visual separando contenido y diseño.

  2. ¿Cuáles son los tres niveles de estilos?

    Inline, Document y Externo.

  3. ¿Qué es un selector de clase?

    Permite aplicar un mismo estilo a varios elementos.

  4. ¿Qué es un selector de id?

    Aplica estilo a un único elemento específico.

  5. ¿Qué es el selector universal?

    Se representa con * y afecta todos los elementos.

  6. ¿Qué son las pseudo-clases?

    Se aplican cuando ocurre una acción, como :hover.

  7. ¿Qué es el modelo de caja?

    Define contenido, padding, border y margin.

Laboratorio 5

  1. Describe Material Design

    Material Design es un sistema de diseño de interfaces de usuario desarrollado por Google. Los diseñadores de UX (experiencia de usuario) lo utilizan para crear experiencias digitales visualmente consistentes, intuitivas y responsivas. Con movimiento integrado, capas y temas dinámicos, Material Design ayuda a los equipos a ofrecer interfaces de usuario (IU) accesibles y de alta calidad en todos los dispositivos y plataformas (En lo que más le invierte una empresa de todos los aspectos de interfaz)
