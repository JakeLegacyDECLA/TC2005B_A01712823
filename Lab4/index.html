Laboratorio 4 - JavaScript

Preguntas Teóricas de JavaScript

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Semejanzas: Ambos usan sintaxis similar basada en C, ambos son orientados a objetos y ambos pueden usarse para desarrollar aplicaciones.

    Diferencias: Java es un lenguaje compilado y fuertemente tipado, mientras que JavaScript es interpretado y débilmente tipado. Java se usa principalmente en aplicaciones backend y móviles, mientras que JavaScript se usa principalmente en desarrollo web.

  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

    Algunos métodos del objeto Date son:

    • getFullYear()
    • getMonth()
    • getDate()
    • getHours()
    • getMinutes()
    • getSeconds()
    • setFullYear()
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5)

    Algunos métodos importantes de los arreglos en JavaScript son:

    • push()
    • pop()
    • shift()
    • unshift()
    • slice()
    • splice()
    • map()
    • filter()
    • reduce()
  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Se declara utilizando las palabras reservadas let o const dentro del bloque de la función. 

    function ejemplo() {
    let variableLocal = 10;
    const otraVariable = 20;
}
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    El uso de variables globales puede generar conflictos de nombres, modificar datos de manera accidental y hacer el código más difícil de mantener. Además, cualquier función puede alterar su valor, lo que puede provocar errores inesperados.
